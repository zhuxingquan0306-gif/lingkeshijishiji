## 🌟 项目简介

灵可世集是一个元宇宙类型的本地生活平台，提供电商、餐饮、住宿、法律等多元化服务。与传统平台不同，我们的推荐机制基于灵可社区的**社区积分**，让真正有价值的服务商获得更多曝光。

### 核心特色

- 🎯 **社区积分驱动** - 基于灵可社区积分的服务商推荐机制
- 🌐 **元宇宙体验** - 沉浸式的数字生活空间
- 🔗 **去中心化** - 基于区块链的透明生态
- 📦 **多元服务** - 涵盖16+个服务领域

## 🚀 快速开始

### 方式一：直接打开

由于本项目是纯静态网站，您可以直接在浏览器中打开 `index.html` 文件。

### 方式二：使用本地服务器

```bash
# 克隆仓库
git clone https://github.com/zhuxingquan0306-gif/lingkeshiji.git

# 进入项目目录
cd lingkeshiji

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 或启动生产服务器
npm start
```

服务器启动后，访问 http://localhost:3000 即可查看网站。

## 🎨 页面详情

### 主页面 (index.html)

主页面包含以下区块：

1. **导航栏** - 动态玻璃效果导航，支持平滑滚动
2. **主视觉区** - 深色背景配粒子动画，展示品牌名称
3. **关于我们** - 平台介绍与核心数据统计
4. **成为服务商** - 展示16种服务商类型
5. **特色功能** - 四大核心功能介绍
6. **社区积分** - 积分机制说明与可视化
7. **加入我们** - 用户注册表单
8. **页脚** - 品牌信息与快速链接

### 世集广场 (marketplace.html)

商家展示页面功能：

- 🔍 **智能搜索** - 支持按商家名称、服务类型搜索
- 📊 **实时统计** - 展示入驻商家数、服务类别、好评率
- 📂 **分类筛选** - 12+服务类别快速筛选
- 📈 **多维排序** - 按热度、积分、评分、时间排序
- 🏪 **商家卡片** - 展示商家信息、评分、积分
- ⭐ **精选推荐** - 社区积分认证的高分商家

## 🛠️ 技术栈

- **HTML5** - 语义化结构
- **CSS3** - 现代样式与动画
  - Flexbox & Grid 布局
  - CSS 变量
  - 响应式设计
  - 自定义动画
- **JavaScript** - 交互逻辑
  - GSAP - 高级动画库
  - ScrollTrigger - 滚动触发动画
  - Canvas API - 粒子效果

### 外部依赖

| 库 | 用途 | CDN |

| GSAP | 动画效果 | jsDelivr |
| ScrollTrigger | 滚动动画 | jsDelivr |
| Inter Tight | 字体 | Google Fonts |

## 🔐 社区积分机制

### 积分获取
- 使用灵可币消费即可获得社区积分
- 积分是灵可世界的权限基础

### 积分用途
- **500+ 积分** - 成为平台服务商
- **积分越高** - 服务曝光机会越多
- **积分兑换** - 享受平台专属权益
